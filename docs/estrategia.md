# Estratégia — Fábrica GilCFP

## Estrutura de Conteúdo por Formato

### Reels Curtos (30-60s) — Instagram
**Principal formato.** Dicas rápidas, ferramentas, mitos, workflows.

**Estrutura de 60 segundos:**
- **Hook (0-3s):** Para o scroll. Frase de impacto. Sem apresentação.
- **Corpo (3-50s):** 2-3 blocos: contexto/problema → solução → resultado concreto
- **CTA (últimos 10s):** "Comenta [PALAVRA] que eu te mando [RECURSO]"

### Carrosséis — LinkedIn
**Autoridade B2B.** Frameworks, comparativos, análises.

### Threads — LinkedIn
**Opinião e análise.** Tendências, opiniões, provocações construtivas.
